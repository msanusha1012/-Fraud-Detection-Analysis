  Bank Fraud Detection Analysis
  
  This project analyzes banking transaction data to identify fraudulent activities using Python (Pandas) for data preprocessing and Power BI for interactive dashboard visualization.

📌 Project Overview

The goal of this project is to explore transaction patterns, identify fraud trends, and create an interactive dashboard that helps financial institutions monitor suspicious activities.

🛠️ Tools & Technologies

Python
Pandas
Power BI
Microsoft Excel

📊 Dashboard Features

Fraud vs Gender Analysis
Fraud by Device Type (ATM, POS, Mobile, Desktop)
Fraud by Merchant Category
Transaction Amount Analysis
State-wise Fraud Analysis
City-wise Transaction Trends
Interactive Filters (Gender, City, Device Type, Merchant Category)

📈 Key Insights

Identified fraud distribution across different device types.
Compared fraud rates between male and female customers.
Analyzed merchant categories with higher fraud occurrences.
Visualized transaction trends across cities and states.
Built an interactive Power BI dashboard for easy data exploration.
